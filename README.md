# Oscillateur harmonique – Simulation et analyse

## Description
Simulation d’un oscillateur harmonique résolue numériquement en Fortran (méthode RK4), puis analyse des données en Python.

## Contenu
- Code Fortran pour la simulation
- Analyse Python avec visualisation
- Ajout de bruit et analyse fréquentielle

## Outils
- Fortran
- Python (NumPy, Matplotlib)

## Objectif
Montrer l’utilisation conjointe du calcul scientifique et de l’analyse de données.
